¿Qué es CSS3?

CSS3 es la última evolución de las hojas de estilo en cascada, ofreciendo mayor flexibilidad en el diseño web.

Principales Características de CSS3

  • Soporte para animaciones y transiciones
  • Uso de flexbox y grid para diseños avanzados
  • Compatibilidad con media queries para diseño responsivo
  • Aplicación de efectos visuales como sombras y bordes redondeados

Ejemplo de Código en CSS3

Un ejemplo de cómo usar **bordes redondeados y sombras** con CSS3:

.box { width: 200px; height: 200px; background-color: #007bff; border-radius: 15px; box-shadow: 4px 4px 10px rgba(0, 0, 0, 0.3); }
